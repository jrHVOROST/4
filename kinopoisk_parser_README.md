 Парсер КиноПоиска

Парсер для получения информации о фильмах через официальное API Kinopoisk.dev.

 Постановка задачи

 Проблема
Необходимость быстрого получения структурированной информации о фильмах из надежного источника без риска блокировок и капчи.

 Решение
Разработка парсера, который:
- Принимает название фильма от пользователя
- Использует официальное API Kinopoisk.dev для получения данных
- Возвращает полную информацию о фильме в структурированном виде
- Сохраняет результаты в форматах JSON и CSV для дальнейшего использования

 Обоснование
Данный проект решает задачу автоматизации сбора киноданных и может быть интегрирован в более крупные системы:
- Рекомендательные сервисы
- Базы данных фильмов
- Аналитические платформы
- Мобильные приложения

 Инструкция по сборке и запуску

 Предварительные требования

- Python 3.7 или выше
- Доступ к интернету
- API ключ от Kinopoisk.dev
